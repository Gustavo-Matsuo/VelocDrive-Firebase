# 🚗 VelocDrive - Car Rental System

# 📌 Sobre o projeto:
Sistema web para aluguel de carros de luxo, com interface desenvolvida utilizando HTML, CSS e JavaScript.

## 🚀 Tecnologias:
- HTML
- CSS
- JavaScript

## ⚙️ Funcionalidades:
- Exibição de veículos disponíveis
- Interface para simulação de aluguel
- Layout responsivo e organizado

## 🎨 Estrutura do projeto:
- HTML → Estrutura da página
- CSS → Estilização
- JavaScript → Interatividade

## ▶️ Como executar:
1. Clonar o repositório  
2. Abrir o arquivo `index.html` no navegador  

## 📈 Objetivo:
Desenvolver habilidades em construção de interfaces web e integração entre HTML, CSS e JavaScript.
